# BcSeoAiAssist プラグイン

baserCMS 5.2用のSEOディスクリプション自動生成プラグインです。

## 概要

OpenAI APIを利用して、ブログ記事や固定ページのSEOディスクリプションを自動生成します。記事本文とキーワードをもとに、検索結果でクリックしたくなる魅力的なディスクリプションを生成します。

## 主な機能

- **AI自動生成**: GPT-4.1-nanoを使用してSEOディスクリプションを自動生成
- **ワンクリック生成**: 編集画面に追加された「AI生成」ボタンで簡単生成
- **キーワード最適化**: 重要度順（左から）にキーワードを考慮
- **最適な文字数**: 約120文字、冒頭40文字で要点を伝える構成
- **対象コンテンツ**: ブログ記事・固定ページに対応

## 必要要件

- baserCMS 5.2以上
- PHP 8.2以上
- OpenAI APIキー（有料）
- BcSeoプラグイン（baserCMS 5.2標準）

## 事前準備

1. インストール後、プラグイン設定画面よりOpenAI APIキーを指定してください
   - 管理画面 > システム管理 > プラグイン管理 > BcSeoAiAssist > 設定
   - OpenAI APIキーを入力して保存

## 使い方

### ブログ記事・固定ページでの利用

1. ブログ記事または固定ページの編集画面を開く
2. 本文とキーワードを入力
3. SEO設定セクションを展開
4. keywords フィールド下の「ディスクリプション生成」ボタンをクリック
5. 生成されたディスクリプションが自動入力される

### 生成のコツ

- **本文を充実させる**: 本文が詳しいほど適切なディスクリプションが生成されます
- **キーワードの順序**: 重要なキーワードを左（先頭）に配置
- **タイトル設定**: タイトルも生成精度向上に寄与します

## 生成条件

- **文字数**: 約120文字（SEOに最適）
- **冒頭強調**: 先頭40文字で最もPRしたい点を伝える
- **誘導重視**: 単なる要約ではなく、クリックを促す表現
- **キーワード**: 自然に組み込まれる（重要度順）
- **安全表現**: 誇張や断定を避けた自然な日本語

## 注意事項

### データ送信について

本プラグインは、記事本文・キーワード・タイトルをOpenAI APIに送信します。
機密情報や個人情報を含む記事での使用には十分ご注意ください。

### APIコストについて

OpenAI APIの利用は従量課金です。使用頻度に応じて料金が発生します。
詳細は[OpenAI Pricing](https://openai.com/pricing)をご確認ください。

### 生成結果の確認

AIが生成したディスクリプションは、そのまま使用せず必ず内容を確認・調整してください。
## 技術仕様

- **使用モデル**: gpt-4.1-nano
- **タイムアウト**: 30秒
- **最大本文長**: 3,000文字（それ以上は切り詰め）
- **API**: REST API (`/baser/api/admin/bc-seo-ai-assist/generate.json`)
